## 프로젝트 소개
- 4인 팀으로 진행된 쿠키런 모작 프로젝트입니다.
- 쿠키런 클라이언트와 7가지의 주요 기능을 수행하는 서버를 구축했습니다.
- 쿠키런 게임 서버는 클라이언트의 API 요청에 대한 인증, 유효성 검사, 응답, 데이터 조회 및 업데이트의 기능을 수행합니다.
<br/>

### 1. 개발 기간
- 게임 서버 개발 기간  : 2024.02.08 ~ 2024.02.28
- 클라이언트 개발 기간 : 2024.02.08 ~ 2024.02.28
<br/>
